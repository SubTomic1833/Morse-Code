# Onekey Morse Code Challenge

Built completely from scratch using standard HTML5 Canvas and vanilla JavaScript.

---

## How to Play

The player who is playing uses the **SPACE BAR** to write out letters of randomly picked words from a list using Morse Code. 

* **Quick Tap:** Registers a **Dot (`.`)**
* **Hold (>200ms):** Registers a **Dash (`-`)**
* **Release & Wait (450ms):** Automatically submits your dots/dashes as a completed letter.

If you correctly input the Morse Code sequence for the letter, it locks in. IF you successfully complete the whole word, you gain 10 points and advance onto the next word. However, if you make a mistake or a typo, then your current progress resets, adn you start from the beginning of the word.

---

## Features

Truely only ONEKEY : You can only use the spaceabar to type out letters in Morse Code.
Live button meter : Shows you a indicator gauge that tracks exactly how long you hold down the spacebar, changing color as soon as you change from a Dot to a Dash.
Clean UI : A clean, black and white UI that matches the colors of the official ONEKEY challenge.
Background easter egg : Easter egg in the background (Hint : decode the morse code in the background) :)
